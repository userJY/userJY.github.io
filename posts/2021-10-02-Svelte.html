    1 Adding bootstrap to sveltekit

    • since sveltekit uses vite, vite automatically allows *.scss imports without manual conversion to css
      • No need to modify vite.config.js nor svelte.config.js
    1. npm install -D sass
    2. npm install bootstrap npm install @popperjs/core(bootstrap js dependency)
    3. add styles.scss to the common lib folder
    4. import $lib/styles.scss in your layout or anywhere really
    5. thatâ€™s it
